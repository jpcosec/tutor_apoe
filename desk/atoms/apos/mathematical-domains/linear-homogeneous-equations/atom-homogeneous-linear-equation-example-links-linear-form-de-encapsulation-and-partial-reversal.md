# Una ELH puede ejemplificar desencapsulacion de forma lineal y reversion parcial

## Respuesta

_Answer the selected 5WH1+ question as one stable knowledge unit._

En el marco de un cuerpo general `K`, una ecuacion lineal homogenea de la forma `ax + by = 0` puede usarse como ejemplo para pensar la relacion entre objeto, desencapsulacion y reversion. Si `L(x,y)=ax+by` se trata como una forma lineal ya encapsulada, el sujeto puede desencapsularla para recuperar el proceso de evaluacion de `L` sobre pares de `K^2`.

Desde alli, la condicion `L(x,y)=0` permite pensar una reversion parcial: no se recupera un unico par, sino un proceso de generacion o caracterizacion de los pares enviados a cero. Cuando se usan inversos multiplicativos, debe explicitarse la restriccion correspondiente: el coeficiente por el que se divide debe ser no nulo. En particular, si `a=b=0`, la ecuacion `ax+by=0` describe todo `K^2`, por lo que no corresponde aplicar el mismo razonamiento de despeje con inversos.

Este ejemplo debe tratarse como una instancia situada dentro de una descomposicion genetica o de un dominio matematico especifico, no como una definicion general de reversion.
