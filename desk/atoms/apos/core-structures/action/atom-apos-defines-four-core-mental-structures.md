# APOS define cuatro estructuras mentales centrales

## Respuesta

El acrónimo APOS significa Acción, Proceso, Objeto y Esquema, y el libro presenta estas cuatro entidades como las estructuras mentales centrales para modelar la construcción del conocimiento matemático.

## Procedencia

Fuente principal: Capítulos 1 y 3 del libro de Arnon et al. (2014), extraído desde `sources/Arnon et al. (2014). APOS-theory-framework-research-curriculum-development-mathematics-education.pdf`.
