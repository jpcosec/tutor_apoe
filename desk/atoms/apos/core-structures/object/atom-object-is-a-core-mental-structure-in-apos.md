# El Objeto es una estructura mental central en APOS

## Respuesta

APOS incluye el Objeto como una estructura mental que permite tratar un proceso matemático como una entidad sobre la cual pueden aplicarse nuevas acciones y razonamientos.

Fuente principal: Capítulo 3, sección 3.2.3, del libro de Arnon et al. (2014), extraído desde `sources/Arnon et al. (2014). APOS-theory-framework-research-curriculum-development-mathematics-education.pdf`.
