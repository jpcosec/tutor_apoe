# Un Proceso permite control mental sin ejecutar cada paso explícitamente

## Respuesta

En APOS, un Proceso es una estructura mental que realiza la misma transformación que una Acción, pero enteramente en la mente, permitiendo imaginar pasos, omitirlos e incluso revertirlos.
