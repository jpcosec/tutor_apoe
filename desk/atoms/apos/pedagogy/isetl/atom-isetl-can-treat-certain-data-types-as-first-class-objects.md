# ISETL puede tratar ciertos tipos de datos como objetos de primera clase

## Respuesta

El libro destaca que ISETL es potente porque puede operar sobre tipos de datos como funcs y tratarlos como objetos de primera clase, incluso devolviendo funciones como resultados.

## Procedencia

Fuente principal: Capítulo 5, sección 5.2.4, del libro de Arnon et al. (2014), extraído desde `sources/Arnon et al. (2014). APOS-theory-framework-research-curriculum-development-mathematics-education.pdf`.
