# Los rotulos de procesos en una descomposicion genetica deben vincularse con el objeto matematico buscado

## Respuesta

_Answer the selected 5WH1+ question as one stable knowledge unit._

En una descomposicion genetica, los procesos no deberian rotularse solo con descripciones genericas de una accion visible, como "calcular la salida", si ese rotulo no muestra su papel en la construccion del objeto matematico que se estudia.

El rotulo debe ayudar a ver como el proceso tributa a la construccion esperada. Por ejemplo, si el objetivo es construir el objeto conjunto solucion de una ecuacion lineal homogenea, es mas informativo hablar de un proceso de evaluacion de una forma lineal o de un proceso de generacion de pares solucion que de un proceso generico de calculo.

Esta precaucion evita confundir una descripcion algoritmica externa con una estructura mental situada dentro de la descomposicion genetica.
