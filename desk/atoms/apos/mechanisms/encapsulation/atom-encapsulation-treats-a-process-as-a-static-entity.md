# La encapsulación permite tratar un proceso como una entidad estática

## Respuesta

La encapsulación ocurre cuando el individuo llega a ver un proceso como una totalidad sobre la que pueden aplicarse transformaciones, convirtiéndolo en un objeto cognitivo.
