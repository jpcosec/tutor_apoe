# La reversion puede operar sobre procesos recuperados por desencapsulacion

## Respuesta

_Answer the selected 5WH1+ question as one stable knowledge unit._

En una descomposicion genetica APOS, la reversion no debe vincularse solamente con la interiorizacion de una accion en un proceso. Tambien puede operar cuando un objeto cognitivo ya construido se desencapsula para recuperar el proceso que le dio origen, y ese proceso recuperado se revierte para construir un nuevo proceso.

Una forma de registrar esta ruta es:

```text
Objeto O -> desencapsulacion -> proceso p1 -> reversion -> proceso p2
```

Esta formulacion mantiene distinguidos los mecanismos: la desencapsulacion recupera el proceso subyacente de un objeto, mientras que la reversion reconstruye ese proceso en sentido inverso para producir otro proceso.
